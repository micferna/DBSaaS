# DBSaaS

Self-hosted Database-as-a-Service platform. Deploy PostgreSQL, Redis, and MariaDB instances in one click with automatic Docker provisioning, TLS encryption, Traefik reverse proxy, and hourly Stripe billing.

## Architecture

| Service | Stack | Port | Role |
|---------|-------|------|------|
| `api/` | Rust (Axum + SQLx + Bollard) | 3001 | Backend API, Docker provisioning, billing |
| `web/` | Next.js 16 (React 19, Tailwind, shadcn) | 3000 | Client dashboard |
| `admin/` | Next.js 16 (React 19, Tailwind, shadcn, recharts) | 3003 | Admin panel |
| `postgres` | PostgreSQL 18 | 5432 | Platform database |
| `redis` | Redis 8 | 6379 | Platform cache |
| `traefik` | Traefik v3.6 | 8880/8443 | Reverse proxy + TLS termination |

## Features

- **One-click database creation** with plan-based resource allocation
- **Multi-server Docker provisioning** via Bollard (mTLS secured)
- **Automatic TLS** certificate generation for client connections
- **Hourly billing** capped at monthly price, Stripe integration
- **Real-time monitoring** with Prometheus metrics and container stats
- **Alerts system** with email notifications (SMTP)
- **Database backups** with scheduled and on-demand support
- **Private networks** for inter-database communication
- **SQL migrations** runner built into the dashboard
- **Invitation system** with codes, max uses, and expiration
- **Admin panel** for managing plans, users, servers, and billing

## Quick Start

```bash
# 1. Generate secrets
./scripts/init-secrets.sh

# 2. Start all services
docker compose up -d

# 3. Access
#    Web client:  http://localhost:3000
#    Admin panel: http://localhost:3003
#    API:         http://localhost:3001
```

## Development

```bash
# API (Rust)
cd api && cargo run

# Web client
cd web && npm install && npm run dev

# Admin panel
cd admin && npm install && npm run dev
```

## Environment Variables

Copy `.env.example` and adjust as needed:

```
PLATFORM_DOMAIN       # Your domain (default: localhost)
PORT_RANGE_START      # Port range for provisioned databases
PORT_RANGE_END
MAX_DATABASES_PER_USER
REGISTRATION_ENABLED  # Open or invite-only registration
```

Secrets are managed via Docker secrets in `./secrets/` (generated by `init-secrets.sh`):
`DATABASE_URL`, `JWT_SECRET`, `ENCRYPTION_KEY`, `STRIPE_SECRET_KEY`, `STRIPE_WEBHOOK_SECRET`

## Project Structure

```
api/
  src/
    handlers/       # HTTP routes (auth, databases, billing, admin, etc.)
    middleware/      # JWT auth, admin guard, rate limiting
    models/         # SQLx structs
    repository/     # SQL queries
    services/       # Business logic (provisioner, billing, TLS, alerts)
    utils/          # Crypto, Docker client, port pool
  migrations/       # Numbered SQL migrations
web/                # Next.js client app (English UI)
admin/              # Next.js admin panel (French UI)
dns/                # Unbound DNS resolver config
traefik/            # Traefik reverse proxy config
nftables-helper/    # Firewall helper container
scripts/            # Setup & utility scripts
```
